# Portfolio Website

Personal portfolio website built with React, TypeScript, and SASS.

## Tech Stack
- React
- TypeScript
- Redux Toolkit
- SASS
- Google Maps API

## Getting Started

```bash
npm install
npm run dev
```

## Deployment
```bash
npm run deploy
```
